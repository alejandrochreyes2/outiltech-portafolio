## CLÁUSULA PRIMERA — OBJETO DEL CONTRATO

EL DESARROLLADOR se obliga a diseñar, desarrollar, implementar y entregar una **plataforma de comercio electrónico (e-commerce)** para EL CLIENTE, bajo el nombre comercial **"Outiltech"**, accesible en la URL https://outiltech.co, que incluye:

1. Tienda web pública con catálogo de 114 productos
2. Sistema de carrito de compras y checkout completo
3. Integración con pasarela de pagos Wompi (tarjeta de crédito/débito y PSE)
4. Sistema de pago manual con Nequi/Daviplata (QR + comprobante)
5. Panel de administración con dashboard en tiempo real
6. Sistema de gestión de pedidos con estados y notificaciones
7. 3 microservicios backend en .NET 8 con API Gateway
8. Base de datos PostgreSQL, MongoDB y Supabase
9. Infraestructura en Hetzner Cloud con Coolify
10. Dominio outiltech.co configurado con Cloudflare
11. Documentación técnica completa (13 documentos)
12. 30 horas de capacitación y soporte post-entrega
